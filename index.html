Solution: I assigned the window.onload property an event handler, a function that handles code in response to an event. The event handler executes as soon as an action fires an event. As a page loads, the browser may render elements, styles, and scripts that require additional time to process. This means that some DOM nodes may not exist until the browser finishes loading the page.
